PWA笔记

PWA全称是：Progressive Web App；渐进式的web应用；完全不影响现有的web应用，但是可以让你渐渐的拥有更加好的体验。

用到的技术

一个普通的web应用要改造成一个PWA，需要用到的技术有：

manifest 应用清单 service worker 服务器与浏览器之间的中间人 promise ES6新增的构造函数 fetch 为web异步获取资源的api caches 主要用于请求的缓存 notification 浏览器通知

先准备一个普通的web应用

开始改造

1、在项目根目录准备一个manifest.json文件

{
	"name":"testrobot",
	"short_name":"robot",
	"start_url":"index.htm",
	"display":"standalone",
	"background_color":"#fff",
	"icons":[{
      "src": "logo.png",
      "sizes": "52x52",
      "type": "image/png"
    },{
       "src": "logo.png",
       "sizes": "144x144",
       "type": "image/png"
    }]
}

注：icons图片要注意，如果空着，后续可能不会出现安装的按钮。

并在我们的web页面中添加这个manifest.json

<link rel="manifest" href="manifest.json">

manifest更多的相关知识可以查看：https://developer.mozilla.org/zh-CN/docs/Web/Manifest

2、判断浏览器是否支持service Worker

3、在sw.js中添加逻辑代码

//加载缓存
const CACHE_NAME='cache_v1'
const resurls=['./','./mh.txt','./bootstrap.min.css','./manifest.json']
self.addEventListener('install',async e=>{
	console.log('开始加载缓存')
	const cache = await caches.open(CACHE_NAME)
	await cache.addAll(resurls)
	await self.skipWaiting()
})
//删除旧缓存
self.addEventListener('activate',async e=>{
	console.log('开始删除旧缓存')
	const keys = await caches.keys()
	keys.forEach(key =>{
		if (key!==CACHE_NAME){
			caches.delete(key)
		}
	})
	await self.clients.claim()
})
//fetch请求
self.addEventListener('fetch',async e=>{
	const req = e.request
	//只缓存同源的内容
	const url = new URL(req.url)
	if (url.origin!==self.origin){
		return
	}
	//如果是api接口，使用网络优先
	if(req.url.includes('/api')){
		e.respondWith(networkFirst(req))
	}else{
		e.respondWith(cacheFirst(req))
	}
	
})

//网络优先
async function networkFirst(req){
	const cache = await caches.open(CACHE_NAME)
	try{
		const fresh = await	fetch(req)
		//获取到数据后，应该放到缓存一份
		cache.put(req,fresh.clone())
		return fresh
	}catch (e){
		const cached = cache.match(req)
		return cached
	}
}

//缓存优先
async function cacheFirst(req){
	const cache = await caches.open(CACHE_NAME)
	const cached = cache.match(req)
	if (cached){
		return cached
	}else{
		const fresh = await fetch(req)
		return fresh
	}
}

fetch更多的相关知识可以查看：https://developer.mozilla.org/zh-CN/docs/Web/API/Fetch_API

cache更多的相关知识可以查看：https://developer.mozilla.org/zh-CN/docs/Web/API/Cache

4、使用弹窗通知

<script>
	//申请弹窗权限
	if(Notification.permission==='default'){
		Notification.requestPermission()
	}
	//检测用户是否断网
	if (!navigator.onLine){
		new Notification('提示',{body:"网络未连接，此时访问的是缓存"})
	}
	//检测网络是否已经联网
	window.addEventListener('online',()=>{
		new Notification('提示',{
			body:"网络已恢复，请刷新访问最新数据"
		})
	})
</script>

notification更多的相关知识可以查看：https://developer.mozilla.org/zh-CN/docs/Web/API/notification
